# 项目功能文档

本文档说明「简历大师」的功能模块和需求。

---

## 一、功能总览

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 简历管理 | 创建、编辑、删除、预览简历 | P0 |
| 模板选择 | 5种简历模板切换 | P0 |
| PDF 导出 | 生成并下载 PDF 文件 | P0 |
| PDF 导入 | 从 PDF 解析简历内容 | P1 |
| 简历分享 | 生成分享链接 | P1 |
| 简历优化 | AI 智能优化简历措辞 | P2 |

---

## 二、功能详情

### 2.1 简历管理

**路由：** `/resumes`

**功能：**
- 显示用户所有简历列表
- 支持搜索、筛选（模板）、排序
- 新建空白简历
- 导入 PDF 简历（自动解析）
- 删除简历（二次确认）

**核心操作：**
- 点击简历卡片进入编辑器
- 悬停显示快捷操作（预览、编辑）

---

### 2.2 简历编辑

**路由：** `/editor/:id?`

**功能：**
- 编辑简历各字段（个人信息、教育经历、工作经历、项目经验、技能、获奖、语言）
- 实时预览简历效果
- 模板切换（5种风格）
- 自动保存（3秒无操作触发）
- 手动保存按钮

**字段：**
| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 个人信息 | 姓名、职位、邮箱、电话、地址等 | 是 |
| 教育经历 | 学校、学位、专业、时间 | 可多条 |
| 工作经历 | 公司、职位、时间、成就 | 可多条 |
| 项目经验 | 项目名、角色、技术栈、描述 | 可多条 |
| 专业技能 | 分类技能列表 | 可多条 |
| 获奖荣誉 | 奖项、时间、颁发机构 | 可多条 |
| 语言能力 | 语言名称、熟练度 | 可多条 |

---

### 2.3 PDF 导出

**功能：**
- 一键生成高质量 PDF
- 支持 A4 纸张格式
- 自动应用当前模板样式

**实现方式：** 前端 `@react-pdf/renderer` 生成

---

### 2.4 PDF 导入

**入口：** 简历列表页 → 导入按钮

**功能：**
- 上传 PDF 文件（拖拽/点击）
- AI 自动解析 PDF 内容
- 解析后进入编辑器，可修改

**解析内容：**
- 个人信息（姓名、职位、联系方式）
- 教育经历
- 工作经历
- 项目经验
- 专业技能

---

### 2.5 简历分享

**入口：** 简历卡片菜单 → 分享

**功能：**
- 生成唯一分享链接
- 复制链接
- 取消分享

**访问：** 无需登录，通过 `/shared/:token` 访问

---

### 2.6 简历优化 ⭐ 新功能

**位置：** 简历编辑器内

**功能：**
- **单条优化**：在工作经历、项目经验等条目的编辑区域，点击 AI 优化按钮
- **一键优化**：顶部工具栏的「一键优化」按钮，一次性优化整份简历

**单条优化流程：**
```
编辑条目 → 点击 Sparkles 图标 → 选择优化类型 → 查看优化结果 → 确认应用
```

**一键优化流程：**
```
点击「一键优化」按钮 → 选择优化类型 → AI 处理整份简历 → 预览优化摘要 → 确认应用
```

**优化类型：**
| 类型 | 说明 |
|------|------|
| 措辞优化 | 使描述更专业、简洁 |
| 关键词增强 | 增强 ATS 关键词 |
| 成就量化 | 量化工作成果 |
| 全部 | 综合优化 |

**技术实现：**
- 前端：OptimizeDialog、FullOptimizeDialog 组件
- 后端：POST /api/v1/resumes/optimize
- API：MiniMax API 调用

---

## 三、页面结构

```
/ (首页)
├── /login (登录)
├── /register (注册)
├── /resumes (简历列表)
│   └── /editor/:id (简历编辑器)
│       └── 简历优化弹窗
└── /shared/:token (分享页，公开)
```

---

## 四、技术要点

1. **状态管理：** React Context (AuthContext)
2. **表单处理：** React Hook Form + Zod
3. **PDF 生成：** @react-pdf/renderer
4. **PDF 解析：** pdfplumber（后端）
5. **AI 优化：** MiniMax API（后端）
6. **认证：** JWT Token
