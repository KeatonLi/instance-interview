# AI 简历优化接口

## 一、接口信息

**路径：** `POST /api/v1/resumes/optimize`

**认证：** 需要

## 二、请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| content | string | 是 | 要优化的简历内容 |
| type | string | 是 | 内容类型：work_experience / project / education / award |
| optimize_type | string | 否 | 优化类型，默认 all |

## 三、响应格式

```json
{
  "code": 0,
  "message": "优化成功",
  "data": {
    "original": "原文内容",
    "optimized": "优化后内容",
    "changes": ["变化说明1", "变化说明2"]
  }
}
```

## 四、优化类型

| 类型 | 值 | 说明 |
|------|------|------|
| 措辞优化 | wording | 使描述更专业、简洁 |
| 关键词增强 | keywords | 增强 ATS 关键词 |
| 成就量化 | quantify | 量化工作成果 |
| 全部 | all | 综合优化（默认） |

## 五、Prompt 设计

### 系统指令

```
你是一个专业的简历优化助手，帮助用户优化简历中的描述。
要求：
1. 措辞专业、简洁、有力
2. 增强相关技术关键词
3. 量化工作成果（使用百分比、具体数字）
4. 保持原意，仅优化表达
5. 每条优化点用一句话说明
```

### 用户输入格式

```
请优化以下{type}描述：

---
{content}
---

优化类型：{optimize_type}
```

## 六、错误处理

| 错误 | code | message |
|------|------|---------|
| 内容为空 | 1 | 请输入要优化的内容 |
| 内容过长 | 1 | 内容不能超过 500 字 |
| AI 服务异常 | 1 | AI 服务暂时不可用，请稍后重试 |
| 未配置 API Key | 1 | AI 功能未配置，请联系管理员 |

## 七、限流

- 单用户每分钟最多 10 次请求
- 返回 429 表示请求过于频繁
